image-compare API

Usage

Pass image_url and other_image_url as query params to run a comparison.

Endpoint: https://image-compare.hcti.io/api

Example

https://image-compare.hcti.io/api?image_url=https://hcti.io/v1/image/28c381f2-ca52-43de-a984-3b17597a1a7b?width=200&other_image_url=https://hcti.io/v1/image/52842f88-a2c0-4277-b1c4-7c07cc0895cc.png
